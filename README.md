## Installation

- Install informasi Composer

```bash
composer install
```

- Copy isian .env.example ke .env

- Menggenerate kunci aplikasi

```bash
php artisan key:generate
```

- Migrasi Database

```bash
php artisan migrate
```

- Seeding Data ke Database

```bash
php artisan db:seed
```

- Instalasi React Depedencies

```bash
npm install
```

```bash
npm run build
```
